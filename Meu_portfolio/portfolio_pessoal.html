Dicas de desenvolvimento

Aprendizados do dia a dia que fazem a diferença. Atualizo sempre que descobrir algo novo.

React

useCallback e useMemo — quando usar de verdade

Não use esses hooks por padrão. Eles têm custo de memória. Só vale otimizar quando há re-renders perceptíveis ou cálculos pesados.

const soma = useMemo(() =>
  numeros.reduce((a,b) => a+b, 0),
  [numeros] // só recalcula quando mudar
);
TypeScript

Type vs Interface — a diferença que importa

Use interface para objetos e quando precisar de extensão via herança. Use type para unions, intersections e tipos primitivos.

type Status = 'ativo' | 'inativo' | 'pendente';

interface Usuario {
  nome: string;
  status: Status;
}
Node.js

Error handling com async/await do jeito certo

Crie um helper utilitário para evitar try/catch repetitivo em todas as rotas assíncronas da sua API.

const asyncHandler = fn => (req, res, next) =>
  Promise.resolve(fn(req, res, next))
    .catch(next);
CSS

Container Queries — o futuro do layout responsivo

Ao invés de adaptar o layout baseado no viewport, adapte com base no tamanho do container pai. Muito mais poderoso para componentes reutilizáveis.

@container (min-width: 400px) {
  .card { grid-template-columns: 1fr 1fr; }
}
Java

Records no Java 16+ — classes imutáveis sem boilerplate

Records geram automaticamente construtor, getters, equals, hashCode e toString. Perfeito para DTOs e Value Objects.

record UsuarioDTO(String nome, String email) {}

// uso:
var u = new UsuarioDTO("Ana", "ana@dev.com");
Git

Commits semânticos salvam vidas (e code reviews)

Conventional commits deixam o histórico legível e habilitam changelogs automáticos. feat, fix, chore, docs, refactor — simples e eficaz.

git commit -m "feat(auth): adiciona login com Google OAuth"
git commit -m "fix(api): corrige timeout nas requisições"
Angular

OnPush Change Detection — performance real

Use ChangeDetectionStrategy.OnPush para evitar renderizações desnecessárias. Angular só atualiza quando o input muda.

@Component({
  selector: 'app-card',
  template: `{{ data.nome }}`,
  changeDetection: ChangeDetectionStrategy.OnPush
})
export class CardComponent {
  @Input() data!: any;
}
CSS

Use Flexbox para layouts simples

Flexbox resolve 80% dos layouts modernos com menos código.

.container {
  display: flex;
  justify-content: space-between;
  align-items: center;
}

Foco & produtividade

Técnica Pomodoro

Trabalhe com foco total por 25 minutos, depois descanse 5. Cada ciclo te deixa mais produtiva.

Hora de focar
25:00
Sessão 1 de 4
0 pomodoros completos hoje ✦ ✦ ✦ ✦
Como funciona?
1. Escolha uma tarefa para focar
2. Clique em iniciar — 25 minutos de foco total
3. Quando o alarme tocar, faça uma pausa de 5 min
4. A cada 4 pomodoros, faça uma pausa longa de 15 min

Esse ciclo aumenta a produtividade e reduz a fadiga mental.
Dica da dev 💜
Durante as pausas, me afasto da tela completamente. Um alongamento rápido, um copo d'água ou só olhar pela janela por 5 minutos faz toda a diferença na concentração dos próximos ciclos.
